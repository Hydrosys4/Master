Getting Started:



Getting started

If you want to try the project I prepared ready to use SD card image with all the required dependencies and SW. (the system is  based on the raspbian jessie lite). It requires an SD card of at least 8Gbyte. The file is quite big around 1.5Gbyte (I was not able to compress more than this)

Below the link to download the file:

https://drive.google.com/open?id=0B3T1HMFoll2GelowSlhaNThjbzQ

After downloading the file you can load it to the SD card and start using the hydrosystem.

Next Step:

Loading the OS in the SD card

 

If you are interested to the code, this is the github link:

https://github.com/Hydrosys4/Master

 

First steps

Now that you have the SD card loaded with the SW, it is time to insert into the Raspberry PI3 and power on.

After few seconds the system will be loaded and if everything has gone right you should be able to see a new WiFi network.

Take your smartphone and  log in to the new network.

Name of the network: hydrosys4



password:
hydrosystem

You should be now connected to the network generated by the raspberry PI.

Now just type the following string to the URL of the web browser in your smartphone or PC:

192.168.0.172:5012

Congratulations, you have entered the hydrosys4 interface.

 

Click on the menu button on the top right corner and select login.

Now enter the default username and password:

username: admin



password: default

I strongly recommend to change the password after you are logged in.
